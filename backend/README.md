# WeatherNow Backend API

WeatherNow is a modern weather forecast application backend built with Node.js and Express. It provides RESTful APIs for accessing weather data from OpenWeatherMap.

## Features

- 🌤️ Current weather data
- 📅 5-day weather forecast
- 🔍 City search with autocomplete
- 📍 Weather by coordinates
- 💨 Air quality data
- 🚦 Rate limiting
- 💾 Response caching
- 🔒 Security headers with Helmet
- 📦 Compression for better performance

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- OpenWeatherMap API key
